# User Management System (Full Stack)

This is a complete User Management application with a React frontend and a Node.js/MongoDB backend. It features a responsive UI, real-time search, status filtering, and CSV export capabilities.

## Project Structure

*   `frontend/`: React application built with Vite.
*   `backend/`: Express.js API with MongoDB integration.

## Quick Start

### 1. Prerequisites
*   Node.js installed
*   MongoDB instance (local or Atlas)

### 2. Backend Setup
1.  Navigate to `/backend`
2.  Create a `.env` file with `MONGODB_URI` and `PORT=5000`
3.  Run `npm install`
4.  Run `npm run dev`

### 3. Frontend Setup
1.  Navigate to `/frontend`
2.  Run `npm install`
3.  Run `npm run dev`

Access the app at `http://localhost:5173`.

## Key Features
*   **CRUD Operations**: Manage user profiles with ease.
*   **Integrated Filtering**: Search and filter by status directly within the table.
*   **Premium UI**: Clean, modern design using CSS Modules and Lucide icons.
*   **Data Portability**: Export your user data to CSV anytime.
*   **Health Checks**: Built-in 10-second server heartbeats.
